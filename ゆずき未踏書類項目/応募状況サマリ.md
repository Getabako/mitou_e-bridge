## 追加必要書類

### 1. 保護者承諾書（法定代理人）- **必須（未成年のため）**
テンプレート: `it2026-koubo-shodakusho_houteidairinin.docx`

渡辺柚気は2007年10月11日生まれ → プロジェクト開始時（2026年6月）は**18歳**
→ 民法上の成人（18歳）だが、高校3年生〜大学1年生の期間
→ **2026年4月1日時点で18歳**（＝成人）のため、法定代理人の承諾書は不要の可能性あり

**確認事項**: 2022年4月1日以降、成年年齢は18歳に引き下げ済み。渡辺柚気は2026年4月1日時点で18歳のため、法律上は成年。ただし、公募要領のフォームで「未成年メンバーがいる→はい」と回答している。

→ 井上・加藤が未成年の場合（2008年4月2日以降生まれの場合）は法定代理人の承諾書が必要。

## 提案プロジェクト詳細資料（10ページ以内）- 内容サマリ

index.htmlに全9セクションが記載済み。Word/PDF化が必要。

| セクション | 内容 | ページ目安 |
|---|---|---|
| ① プロジェクトの背景・課題認識 | 1,000万人のコミュ困難、既存アプローチ限界、eスポーツとの出会い | 1.5p |
| ② 提案プロジェクトの概要・解決策 | 3つの柱、プログレッション、VALORANT選定理由、技術新規性 | 2p |
| ③ 開発の具体的な進め方 | MVP定義、システム構成、スケジュール、3名体制、リスク対策 | 2p |
| ④ 提案者の実力・能力を示す実績 | 当事者×開発者融合、チーム紹介、PM体制、PoC結果 | 1.5p |
| ⑤ 事業化の新規性・優位性・市場規模 | 競合分析、参入障壁、TAM/SAM/SOM、ビジョン | 1.5p |
| ⑥ 事業化の具体的な進め方 | ビジネスモデル（B2C/B2B）、集客戦略、ロードマップ | 1p |
| ⑦ 事業期間終了後の計画 | 法人化、人員計画、売上計画、資金調達 | 1p |
| ⑧ 予算内訳 | 3名の稼働計画・時間単価・内訳（合計3,024,000円） | 1p |
| ⑨ 未踏性・独創性 | パラダイムシフト、インクルーシブAI、空白地帯、社会的インパクト | 1p |

**注意**: 公募要領の記入項目は①〜⑧の8項目だが、HTMLには⑨まである。提出時は公募要領の項目に合わせて再構成が必要:
1. なにをつくるか → ①②に対応
2. どんな出し方を考えているか → ⑥に対応
3. 斬新さの主張、期待される効果など → ⑤⑨に対応
4. 具体的な進め方と予算 → ③⑧に対応
5. 提案者の腕前を証明できるもの → ④に対応
6. プロジェクト遂行にあたっての特記事項 → 未成年、所属変更等
7. IT以外の勉強、特技、生活、趣味など → **要追加**（eスポーツ部活動、吃音との向き合い等）
8. 将来のITについて思うこと・期すること → **要追加**
